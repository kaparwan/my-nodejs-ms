# NodeJS - Example Microservice Project Structure

## To run project execute below two commands
- docker-compose build
- docker-compose up


#### Clean and Delete node_nodules
```
npm run clean
```

### Lint and Fix
```
npm run lint
```

### Run Unit Tests
```
npm run test:unit
```

### Test and Generate Coverage
```
npm run test
```

### Commit
```
npm run commit
```
